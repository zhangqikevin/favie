Goal
Build a fast, responsive, single‑page marketing website that sells a done‑for‑you, AI‑powered service for US restaurants. This is NOT SaaS; clients don’t operate dashboards. We do the work, they approve and see results.

Tech & Structure

Stack: semantic HTML5 + modern CSS (flex/grid) + vanilla JS for light interactions. No heavy frameworks.
Files: index.html, styles.css, script.js, assets folder with placeholder images/icons (use Unsplash URLs).
Mobile‑first, responsive across breakpoints (375/768/1024/1440).
Accessibility: proper landmarks, alt text for images, keyboard focus styles, color contrast AA.
SEO: title, meta description, Open Graph/Twitter cards, structured data (Organization + Service).
Visuals: warm, appetizing palette (e.g., terracotta #C66C3A, sage #87A78B, cream #F7F3EE, charcoal #2A2A2A), generous white space, soft shadows, subtle parallax or scroll‑reveal animations (CSS/JS).
Typography: Pairing like Inter (UI) + Playfair Display (headlines) from Google Fonts.
Icons: inline SVG or Feather Icons.
CTAs: sticky header “Book a 15‑min Intro Call”, secondary “See Sample Posts”.
