Please redesign the individual agent pages to reduce text density and make them more visual.

Current problem:
The agent pages rely too heavily on long text sections.
They explain the agents clearly, but the pages feel too text-heavy and not visual enough.

Goal:
Make each agent page feel more like a product experience and less like a long-form services page.

New direction:
Use a better mix of:
- short text
- structured cards
- visual previews
- supporting images
- metric blocks
- example outputs

Important:
Do not remove the content meaning.
Instead, convert part of the explanation into visual components.

What to improve on every agent page:

1. Reduce long text blocks
- Shorten paragraphs where possible
- Replace long bullet sections with cards, tags, or grouped modules
- Keep only the most important intro copy

2. Add more visual cards
Use cards to represent:
- key responsibilities
- example outputs
- business signals
- recommendations
- workflows
- capability groups

Instead of showing everything as long bullet lists, break content into smaller visual units.

3. Add relevant supporting images or previews
Each agent page should include visuals that fit the agent’s role.
These do not have to be heavy decorative images.
They can be:
- UI-style preview cards
- mock screenshots
- food thumbnails
- content thumbnails
- rating / review cards
- mini charts
- promotion previews
- creator profile previews

4. Use a richer page structure
Each agent page should feel more layered, for example:
- hero/introduction
- capability cards
- expectation/results cards
- visual examples
- small dashboard snippets
- CTA / learn more / start chat section

7. Keep the pages practical, not decorative
The visuals should support understanding and reduce text fatigue.
Do not add random stock images just to fill space.
Use visuals that feel connected to the work each agent does.

8. UX goal
Each agent page should feel:
- easier to scan
- more visual
- more product-like
- more AI-native
- less like a long explanation page

Important:
The goal is not just to make the page prettier.
The goal is to replace some of the text with useful visual structure so users can understand each agent faster.
